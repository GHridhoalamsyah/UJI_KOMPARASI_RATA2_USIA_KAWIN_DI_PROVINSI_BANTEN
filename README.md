# UJI_KOMPARASI_RATA2_USIA_KAWIN_DI_PROVINSI_BANTEN
Uji Komparasi untuk mengetahui apakah ada perbedaan signifikan rata-rata usia kawin pertama pada kabupaten dan kota provinsi banten pada tahun 2019 dan 2023


METODE YANG DIGUNAKAN
1. UJI MANN WHITNEY U TEST = untuk mengetahui apakah ada perbedaan signifikan rata2 usia kawin pertama di kabupaten dan kota
2. Uji WILCOXON SIGNED RANK TEST = untuk untuk mengetahui apakah ada perbedaan signifikan rata2 usia kawin pertama pada tahun 2019 dan 2023
